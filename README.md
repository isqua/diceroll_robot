# diceroll_robot

This is a source code of a telegram bot: https://t.me/diceroll_robot

## Development

Install dependencies:

```
npm install -g pnpm@7.8
pnpm install
```

Run tests:

```
npm test
```

Run the bot while watching sources:

```
export BOT_TOKEN=<token_obtained_from_botfather>
npm run dev
```
